# bundle-win-zapret2
Терминальная сборка bol-van/zapret2 для Windows

## Оригинальный репозиторий пренадлежит bol-van
[bol-van/zapret2](https://github.com/bol-van/zapret2)

## Как запустить

Для запуска используйте zapret2.bat

## Как редактировать конфигурацию

Конфигурация находиться в файле config.txt.
Чтобы добавить/редактировать порты в правилах, предварительно прописывается захват для tcp и udp `wf-tcp-out` и `wf-udp-out` соответственно.
Далее заполняются правила, не забывая разделять их `new` в конце.
Правила записываются также, как и в версии для Linux.
